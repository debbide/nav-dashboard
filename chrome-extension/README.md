# 导航仪表盘助手 - Chrome 扩展

通过右键菜单快速添加网页到导航仪表盘。

## ✨ 功能特性

- 🖱️ **右键菜单** - 在任意网页右键点击，选择"添加到导航仪表盘"
- 📌 **点击图标** - 或直接点击工具栏图标添加当前页面
- 🔄 **自动填充** - 自动获取页面标题、URL 和图标
- 📁 **分类选择** - 支持选择已有分类
- 🔐 **密码保护** - 支持配置管理密码

## 📦 安装方法

### 方式一：开发者模式加载

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 打开右上角的 **"开发者模式"** 开关
3. 点击 **"加载已解压的扩展程序"**
4. 选择 `chrome-extension` 目录
5. 扩展安装完成！

### 方式二：Edge 浏览器

1. 打开 Edge，访问 `edge://extensions/`
2. 打开 **"开发人员模式"**
3. 点击 **"加载解压缩的扩展"**
4. 选择 `chrome-extension` 目录

## ⚙️ 配置

首次使用前需要配置 API 地址：

1. 点击扩展图标旁的 **"⚙️ 设置"** 按钮
2. 输入你的导航仪表盘地址，例如：
   - 本地：`http://localhost:3000`
   - 远程：`https://nav.example.com`
3. 如果启用了密码保护，输入管理密码
4. 点击 **"保存设置"**
5. 点击 **"测试 API 连接"** 验证配置

## 🚀 使用方法

### 右键菜单添加

1. 在任意网页空白处 **右键点击**
2. 选择 **"添加到导航仪表盘"**
3. 在弹窗中确认信息并选择分类
4. 点击 **"添加"**

### 点击图标添加

1. 点击浏览器工具栏的扩展图标 📌
2. 自动填充当前页面信息
3. 选择分类并点击 **"添加"**

## 🔧 常见问题

**Q: 右键菜单没有出现？**  
A: 刷新页面后重试，或在 `chrome://extensions` 中重新加载扩展。

**Q: 提示 "请先配置 API 地址"？**  
A: 点击设置按钮，输入你的导航仪表盘地址。

**Q: 添加失败，提示网络错误？**  
A: 检查 API 地址是否正确，确保导航仪表盘正在运行。

**Q: 密码验证失败？**  
A: 在设置中检查密码是否正确。
